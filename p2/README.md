#Prácitca 2.

###Requisitos
```bash
sudo apt-get install build-essential doxygen
```

###Compilación

Para compilar:
```bash
make
```

Para limpiar ejecutables:
```bash
make clean
```

###Generación de documentación

Para generar la documentación:
```bash
doxygen Doxyfile
```
